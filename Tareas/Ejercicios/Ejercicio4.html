RAM La RAM Random Access Memory o memoria de accesoaleatorio se utiliza para almacenar temporalmente datos temporales y programas que el procesador necesita acceder rapidamente

SSD La Solid State Drives son un tipo de dispositivos de almacenamiento de datos que utiliza memoria flash e lugar de discos magneticos como los discos duros tradicionales

Procesador El procesador o tambien conocido como CPU Unidad Central de Procesamiento, es el componente fundamental de una computadora que se encarga de ejecutar las instrucciones de los programas y de coordinar las operaciones de los demas componentes del sistema.
